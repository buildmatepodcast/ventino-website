ABOUT THE .CDR FILE
────────────────────────────────────────────────

You asked for a CorelDRAW (.cdr) file. An honest note:

.cdr is a closed, proprietary format owned by Corel. It can only
be *written* by CorelDRAW itself — no other software can create a
genuine .cdr file. So a true .cdr cannot be generated here.

What is provided instead — and what every print vendor and
designer uses for exactly this purpose — is the EPS file in the
/Logo_Vector_EPS/ folder.

  • EPS is the universal vector exchange format.
  • CorelDRAW opens EPS files directly and natively.
  • Once open in CorelDRAW, simply choose
        File ▸ Save As ▸ CorelDRAW (.cdr)
    and you have a native .cdr in seconds.

The EPS (and the PDF and SVG) are fully editable vectors — every
line, the frame, the sill, the wordmark — at infinite scale.

TYPEFACE
────────────────────────────────────────────────

Wordmark and headings:  Cormorant Garamond
    A free, open-source high-contrast display serif.
    Download: fonts.google.com/specimen/Cormorant+Garamond

Captions and specifications:  Helvetica Neue / Arial

In the logo files the wordmark is drawn as live text. If a vendor
does not have Cormorant Garamond installed, ask for the wordmark
to be converted to outlines/curves, or supply the font file.

For a step beyond — Canela or Lyon (licensed) — offer even more
refinement, if budget allows.

A note on the mark

Every line was measured against one sentence:
"A threshold held with poise — an opening that protects,
a frame around the light, calm enough to stay closed
and confident enough to open."

If a future change does not serve the threshold, it does
not belong.
